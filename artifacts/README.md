# Lab Artifacts

This directory stores research artifacts contributed through git by Spark Lab members.

Each artifact should live in its own folder and include:

- `metadata.yml`
- `README.md`
- one preview image or example output
- scripts and small supporting files as needed

Recommended layout:

```text
artifacts/
  artifact-slug/
    metadata.yml
    README.md
    preview.png
    scripts/
    data/
```

Use the `_template/` folder as a starting point for new contributions.
