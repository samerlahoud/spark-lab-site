# LoRaWAN Measurement Campaigns in Lebanon

## Overview

This artifact corresponds to the open Zenodo dataset used to derive path-loss models for LoRaWAN in Lebanon and to evaluate deployment coverage in real environments.

## Contents

- public Zenodo dataset
- measurement documentation
- path-loss model function
- related publication
- preview figure corresponding to Figure 11(a) in the paper

## Related paper

R. El Chall, S. Lahoud, M. El Helou, *LoRaWAN Network: Radio Propagation Models and Performance Evaluation in Various Environments in Lebanon*.

PDF: https://samer.lahoud.fr/pub-pdf/jiot-19.pdf
